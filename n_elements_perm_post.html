A Visual Representation for the Permutations of an $n$ Element Set

A well known result in probability and combinatorial analysis is that the number of unique permutations of an $n$ element set is $n!$. One might wonder how exactly the factorial finds its way into this problem. A short visual representation is provided here.

In a proof-by-induction fashion, consider the following simple 3 element set $S$, consisting of the integers 1, 2 and 3 \[ S=\{1,2,3\} \] All possible permutations of $S$ are given by \[ S=\{1,2,3\} \] \[ S=\{1,3,2\} \] \[ S=\{2,1,3\} \] \[ S=\{2,3,1\} \] \[ S=\{3,1,2\} \] \[ S=\{3,2,1\} \] which reveals something almost immediately. All permutations have been listed in such a way that the first element is fixed, and the remaining two elements are being cycled through the two different permutations of a 2 element set. This shows we can form all permutations of a set by fixing an element, and cycling through all permutations of the set formed by the remaining elements.

Considering the example above, there are two permutations of a two element set, and three possible values for the single fixed element in the three element set. This means there are $3\times2$ total permutations, since each time an element is fixed (where there are three total values to fix) there are two permutations of the remaining elements of the set.

More generally, denoting the number of permutations of a set containing $n$ elements as $P_{n}$, it can be stated that \[ P_{n}=nP_{n-1} \] since the above example has identified such a relationship. Using recurrence to derive an even more general form gives \begin{align*} P_{n} & =n(n-1)P_{n-2}\\ & =n(n-1)(n-2)P_{n-3}\\ & =n(n-1)(n-2)(n-3)P_{n-4} \end{align*} where the identity $P_{n}=nP_{n-1}$ is simply being applied each time. Since $P_{1}=1$ i.e. there is a single permutation for a single element set, for any value of $n$ the above can be simplified to \begin{align*} P_{n} & =n(n-1)(n-2)...(3)(2)P_{1}\\ & =n(n-1)(n-2)...(3)(2)(1)\\ & =n! \end{align*} which proves the well known result for the number of permutations of an $n$ element set.
